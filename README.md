# terrel-test-org/.github

This repository serves as the central governance hub for the **terrel-test-org** organization, providing organization-wide configurations and serving as an access point for reusable Copilot extensions.

## Purpose

- **Organization Governance**: Contains shared configurations, templates, and policies that apply across all repositories in the organization
- **Copilot Extensions Hub**: Central location to pull and share plugins, skills, and custom agents created over time

## What's Inside

### 📋 Copilot Instructions
Organization-wide coding standards and guidelines in `.github/copilot-instructions.md`, including:
- Design system conventions
- Required component mappings
- Styling rules and code patterns
- Anti-patterns to avoid

### 🛠️ Skills
Reusable Copilot skills available in `.github/skills/`:

| Skill | Description |
|-------|-------------|
| **readme-blueprint-generator** | Analyzes project documentation and generates comprehensive README files |
| **training-documentation-writer** | Creates user guides, API documentation, labs, and technical manuals |

## Usage

Skills and configurations from this repository are automatically available to repositories within the organization. Reference them directly in your Copilot workflows to leverage shared tooling and standards.

## Contributing

Feel free to add new skills, custom agents, or plugins to this repository to share them across the organization
